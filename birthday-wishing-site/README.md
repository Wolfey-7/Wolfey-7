# Birthday Wishing Site

Open `index.html` in your browser to use the page. Enter a name and click "Send Wishes" to display a personalized greeting and trigger confetti.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `.github/workflows/pages.yml`

## Run locally
1. Open `index.html`, or
2. Serve with Python:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages deployment
1. Create a GitHub repository and push this folder.
2. Use `main` or `master` branch.
3. The workflow at `.github/workflows/pages.yml` will deploy automatically on push.

Your site will be available at `https://<your-username>.github.io/<repository-name>/`.
